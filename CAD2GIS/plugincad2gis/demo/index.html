Turning legacy CAD drawings into a verifiable GIS system.

A deterministic pipeline that parses historical DWG/DXF communications drawings — graphics, attributes and topology — and warehouses them as a QGIS-ready GeoPackage, measured against ground truth on every dimension. No black boxes: every feature traces back to the CAD entity it came from.

Accuracy, measured — not asserted

Six dimensions scored against an independently-anchored benchmark. Dimensions without an independent ground-truth source are shown but marked not scored, never faked.

Per-feature correctness, independently checked

Each converted feature is verified by a signal that does not come from the classifier's own rule path — so this is a real correctness measurement, not a tautology. Manholes are cross-checked against surveyed coordinate labels; cables against topology; duct symbols against their geometry fingerprint.

From 22,000 mixed entities to a clean comms network

The source drawing interleaves comms, power, water and survey base across 150+ layers. The pipeline scopes to comms, then removes symbol/annotation noise misread as cable.
